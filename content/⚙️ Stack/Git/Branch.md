
#### Show all branch 
```
git branch -a
```

#### Show all branch with hash ref
```
git show-ref
```

#### Create new branch
```
git checkout -b ＜new-branch＞
```

#### Show current branch 
```
git rev-parse --abbrev-ref HEAD`
```

Move to new branch
```
git checkout -b main
```

#### Switching branches
```
git checkout ＜branchname＞
```
